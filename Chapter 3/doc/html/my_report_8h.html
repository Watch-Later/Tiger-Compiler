int errorReport ( )

An alternative to yyerror() used to output error information when an error is encountered on lexical analysis.

Returns
On success, the total number of characters written is returned. On failsure, a negative number is returned.
Remarks
Users can choose either yyerror() or this function to report error. By default, the report is outputed to the standard output.

Parameters

Returns
If the function succeeds, the return value is the outputed character count. If the function fails, the return value is -1.
