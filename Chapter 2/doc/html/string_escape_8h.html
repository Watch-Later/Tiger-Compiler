Translate escape characters.

Parameters

Returns
A translated string.
Remarks
The return string is an new allocated string which internally use makeMemoryBlock(). So, if one attempts to free it, use freeMemoryChain().
Warning
This function only do translations, the caller make sures the escape characters contained is valid.
