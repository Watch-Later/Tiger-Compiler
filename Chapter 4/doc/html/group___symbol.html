mySymbol MySymbol_MakeSymbol ( myString  str)

Make s symbol from a string.

Parameters
[in]strThe string to make symbol from.
Returns
If the string is NULL or empty, return NULL; If the string is valid, return a symbol made from it.
Remarks
When a new symbol is made, it is also added to the internal symbol table to avoid remaking. If the symbol made is already existed, this function simply returns it.
